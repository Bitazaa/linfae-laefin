    เพิ่มบัญชีธนาคาร

    📘 คู่มือตั้งค่า LINE แจ้งเตือนแบบละเอียด

    มือใหม่ สำคัญ
    เริ่มตั้งค่า LINE แจ้งเตือนใน 4 ขั้นตอนหลัก
    หน้านี้เป็นจุดเริ่มต้นแบบสั้น ๆ ก่อน จากนั้นกด Start Setup เพื่อดูขั้นตอนละเอียดทีละหัวข้อ
    1. สร้าง LINE Official Account
    2. เปิดใช้งาน Messaging API
    3. ออก Channel Access Token แล้วนำมาใส่ระบบ
    4. หา Group ID / User ID แล้วทดสอบส่งข้อความ
    คู่มือทีละขั้น ระบบจะจำหัวข้อที่เปิดล่าสุดให้อัตโนมัติ

    LINE แจ้งเตือนต้องใช้ 2 อย่างร่วมกัน:

    • LINE Official Account = บัญชีร้านที่ลูกค้า/ทีมงานเห็น
    • Messaging API = ระบบส่งข้อความอัตโนมัติ

    สำคัญ: ต้องใช้ Channel Access Token ไม่ใช่ Channel Secret

    1. เข้า https://manager.line.biz/
    2. Login ด้วย LINE ส่วนตัว
    3. กดสร้างบัญชี
    4. ตั้งชื่อร้าน
    5. เลือกหมวดหมู่ เช่น ร้านอาหาร
    6. กดสร้างบัญชี
    1. เข้า LINE Official Account Manager
    2. เลือกบัญชีร้าน
    3. ไปที่ ตั้งค่า
    4. ไปที่ Messaging API
    5. กด “ใช้ Messaging API”
    6. ถ้าระบบให้สร้าง Provider ให้ตั้งชื่อ เช่น FoodOrder System
    7. กดยืนยัน
    1. เข้า https://developers.line.biz/console/
    2. เลือก Provider
    3. เลือก Channel ประเภท Messaging API
    4. เข้าแท็บ Messaging API
    5. เลื่อนลงหา “Channel access token”
    6. กด Issue
    7. Copy token มาใส่ช่อง Channel Access Token ในระบบ
    ห้ามใช้ Channel Secret แทน Token
    Token มักยาวมาก และเป็นข้อมูลลับ ห้ามส่งให้คนอื่น
    1. เข้า LINE Developers Console
    2. เข้า Channel > Messaging API
    3. หา “Allow bot to join group chats” และตั้งเป็น Enabled
    4. แนะนำปิด Auto-reply messages
    5. แนะนำปิด Greeting messages

    Webhook ใช้เพื่อรับ event ตอนมีคนพิมพ์ในกลุ่ม ระบบนี้ใช้ Webhook แค่เพื่อหา Group ID เท่านั้น ส่วนการแจ้งเตือนจริงใช้ Push Message

    1. Copy Web App URL ของระบบ (ตัวอย่างด้านล่าง)
    2. ใส่ช่อง Webhook URL
    3. กด Verify
    4. เปิด Use webhook = Enabled
    5. ต้องใช้ /exec ไม่ใช่ /dev
    6. หลังแก้ Code.gs ต้อง Deploy New Version เสมอ
    https://script.google.com/macros/s/xxxxx/exec

    User ID ใช้ส่งแจ้งเตือนเข้าหาคนเดียว และมักขึ้นต้นด้วย U

    1. เข้า LINE Developers Console
    2. เข้า Channel > Messaging API
    3. เลื่อนหา Your user ID
    4. Copy มาใส่ Target ID
    5. เลือก Target Type = user

    Group ID ใช้ส่งเข้ากลุ่ม และมักขึ้นต้นด้วย C เหมาะกับกลุ่มครัว/กลุ่มร้าน

    1. เปิด Allow bot to join group chats
    2. เพิ่ม LINE OA เข้ากลุ่ม LINE
    3. ตั้ง Webhook URL แล้วเปิด Use webhook
    4. พิมพ์ข้อความในกลุ่ม 1 ครั้ง เช่น “ทดสอบ”
    5. กลับมาที่ระบบ แล้วกด “ดึง Group ID ล่าสุด”
    6. ระบบจะเติม Target ID ให้อัตโนมัติ
    7. เลือก Target Type = group
    8. กดบันทึก
    1. เปิดใช้งาน LINE = ON
    2. ใส่ Channel Access Token
    3. เลือก Target Type
    4. ใส่ Target ID
    5. กดบันทึก
    6. กด “ทดสอบส่ง LINE”
    7. ถ้า LINE กลุ่มได้รับข้อความ แปลว่าสำเร็จ
    A: ไม่ใช่ คนละตัวกัน ต้องใช้ Channel Access Token เท่านั้น
    A: แปลว่า OA อยู่ในกลุ่มแล้ว แต่ Webhook อาจยังไม่เข้า ให้เช็ก Webhook URL, Use webhook, Deploy New Version และดู Apps Script Executions
    A: ตรวจว่า Webhook URL เป็น /exec, Web App เปิด Anyone can access, และมี doPost(e)
    A: ต้องเข้า Channel ประเภท Messaging API ไม่ใช่ LINE Login
    A: เช็ก Token, Target ID, Target Type, OA อยู่ในกลุ่ม, และ Allow bot join group เปิดแล้ว
    A: ไม่ต้อง สำหรับระบบส่งแจ้งเตือนใช้ Channel Access Token เท่านั้น
    A: ถ้าหา Group ID แล้ว การส่งแบบ Push Message ไม่จำเป็นต้องพึ่ง Webhook แต่เปิดไว้ได้
